# 🏢 PrimeTech EMS - Employee Management System

**PrimeTech EMS** is a robust, production-ready web application for human resource management. Built with **ASP.NET Core MVC**, it features a secure **Role-Based Access Control (RBAC)** system and is fully deployed for live use.

## 🔗 Live Demo & Deployment
* **Live Website:** [ems-primetech.runasp.net](http://ems-primetech.runasp.net)
* **Architecture:** N-Tier (PL, BLL, DAL)

## 🚀 Key Features
* **Role-Based Access Control (RBAC):** Dedicated **User & Role Controllers** for managing administrative permissions and secure access.
* **Employee & Department Management:** Full lifecycle tracking with complex business logic handled in the BLL.
* **Attachment Service:** Secure file handling and storage for employee-related documents.
* **Identity Framework:** Advanced security using Microsoft Identity with customized user management.
* **Production Ready:** Fully configured for cloud hosting with remote SQL Server integration.

## 🏗️ Project Architecture (3-Tier Model)
1. **PrimeTech.EMS.PL (Presentation Layer):** MVC Views & Controllers (including Auth, Roles, and Users).
2. **PrimeTech.EMS.BLL (Business Logic Layer):** Application services and DTOs.
3. **PrimeTech.EMS.DAL (Data Access Layer):** EF Core Context, Migrations, and SQL Server Persistence.

## 🛠️ Technical Stack
* **Backend:** ASP.NET Core 8.0 MVC
* **Security:** Microsoft Identity (Roles & Users management)
* **Database:** Remote MSSQL Server (databaseasp.net)
* **Hosting:** SmarterASP.NET / runasp.net
